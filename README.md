# CharaROM Device Manifests

A collection of local manifests used for building CharaROM for select devices.

Devices supported:
`dre` - OnePlus Nord N200 5G - Maintained by Chara
`milanf` - Motorola Moto G Stylus 5G (2022) - Maintained by Chara
`veux` - Redmi Note 11 Pro 5G / POCO X4 Pro 5G - Maintained by JPB
`ginkgo` - Redmi Note 8 - Maintained by JPB
`a52sxq` - Samsung Galaxy A52s 5G (SM-A528B, SM-A528N, SM-A528B/DS) - Maintained upstream, built by Chara

Devices in repo that aren't supported (yet):
`pdx237` - Sony Xperia 5V - maintained upstream by eepymeowers/Nora, no builds yet
`salaa` - Realme 7/Narzo 20 Pro/Narzo 30 4G - manifest in progress
`guacamoleb` - OnePlus 7 - manifest maintained by eepymeowers/Nora, no builds yet

More devices will eventually be added here...
